# Remedies Journal

Веб-приложение для учёта приёма гомеопатических препаратов.  
Хранит данные на Яндекс.Диске, имеет удобный интерфейс для телефона, позволяет добавлять события и экспортировать записи в Excel.

## Возможности
- Добавление, редактирование и удаление записей о приёме (дата, препарат, потенция).
- Добавление событий (дата, время, описание) к каждой записи.
- Автоматическое форматирование даты и времени при вводе (маски Cleave.js).
- Хранение данных в JSON на Яндекс.Диске.
- Экспорт в Excel с цветовым оформлением колонок.

## Технологии
- Python + Flask
- Яндекс.Диск API (yadisk)
- Bootstrap 5
- Cleave.js (маски ввода)
- OpenPyXL (генерация Excel)

## Локальный запуск
1. Клонируйте репозиторий:

```bash
git clone https://github.com/ваш-логин/remedies-journal.git
cd remedies-journal
```
2. Установите зависимости:

```bash
pip install -r requirements.txt
```

3. Создайте файл `.env` и укажите токен **Яндекс.Диска**.

4. Запустите приложение:

```bash
python app.py
```

5. Откройте http://127.0.0.1:5009.

## Переменные окружения

- YANDEX_TOKEN – токен доступа к Яндекс.Диску
- REMOTE_PATH – путь к файлу на диске (например, /RemediesJournal/data.json)
- LOCAL_FALLBACK – имя локального файла для резервного копирования
- SECRET_KEY – секретный ключ Flask
